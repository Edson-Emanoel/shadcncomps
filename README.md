# Desafio · Props Tipadas em Cards com shadcn/ui

**Tech Stack da aula**

- Next.js 16.0.1 (App Router)
- React 19.2.0
- TypeScript 5
- Tailwind CSS 4
- shadcn/ui + Radix UI + cva + clsx + tailwind-merge
- lucide-react 0.553.0

**Desafio da aula:** criar Cards totalmente tipados com TypeScript — incluindo variantes (size, variant), ícones dinâmicos, children, eventos onClick tipados e integração perfeita com cva + tailwind-merge — tudo sem usar any
